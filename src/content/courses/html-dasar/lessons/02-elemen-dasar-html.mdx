Setelah tahu apa itu HTML, sekarang saatnya kenalan lebih dekat sama elemen-elemen dasarnya. Ini ibarat kamu kenalan sama anggota keluarga besar HTML. Ada yang cerewet kayak `<h1>`, ada yang simpel kayak `<br>`, dan ada juga yang fleksibel kayak `<div>`.

## Heading (Judul)

Heading dipakai untuk memberi judul atau subjudul. HTML punya 6 level heading:

```html
<h1>Judul Utama</h1>
<h2>Subjudul</h2>
<h3>Sub-subjudul</h3>
... sampai <h6>
```

Semakin besar angkanya, semakin kecil ukurannya.

## Paragraf

Gunakan tag `<p>` untuk membuat paragraf:

```html
<p>Ini adalah satu paragraf teks.</p>
```

## Gambar (Image)

Untuk menampilkan gambar, gunakan tag `<img>`:

```html
<img src="/images/kucing.jpg" alt="Foto kucing lucu" />
```

- `src` = lokasi file gambar
- `alt` = deskripsi gambar (untuk aksesibilitas dan SEO)

## Tautan (Link)

HTML pakai tag `<a>` (anchor) untuk membuat link:

```html
<a href="https://google.com">Kunjungi Google</a>
```

- `href` = alamat tujuan link

## Garis Horizontal dan Baris Baru

- `<hr />` = garis horizontal, misalnya buat pemisah antar bagian
- `<br />` = baris baru (kayak pencet Enter)

```html
<p>Halo<br />Dunia!</p>
<hr />
```

## Daftar (List)

### Ordered List (Berurutan)
```html
<ol>
  <li>Bangun tidur</li>
  <li>Mandi</li>
  <li>Ngoding</li>
</ol>
```

### Unordered List (Tanda titik)
```html
<ul>
  <li>Kopi</li>
  <li>Teh</li>
  <li>Air putih</li>
</ul>
```

## Div dan Span

- `<div>` digunakan sebagai pembungkus blok
- `<span>` untuk teks dalam satu baris (inline)

```html
<div>
  <p>Ini dibungkus dalam div.</p>
</div>

<p>Ini <span style="color:red">kata penting</span> dalam kalimat.</p>
```

## Latihan

1. Buat file `latihan.html`
2. Tambahkan heading, paragraf, gambar, dan tautan ke situs favoritmu
3. Tambahkan daftar kegiatan harianmu
4. Coba buat garis horizontal sebagai pemisah
