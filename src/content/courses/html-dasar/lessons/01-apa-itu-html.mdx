## Kenapa HTML Penting?

HTML adalah *pondasi* dari semua website. Bahkan website yang super canggih sekalipun, pada dasarnya tetap butuh HTML untuk:

- Menyusun teks, gambar, dan video
- Membuat tautan antar halaman
- Menyusun form input seperti login, pencarian, dll

Tanpa HTML, website itu cuma... angan-angan.

## Struktur Dasar HTML

Berikut ini contoh struktur dasar dokumen HTML:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Halo Dunia</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
  </body>
</html>
```

### Penjelasan:

- `<!DOCTYPE html>` memberi tahu browser bahwa ini adalah dokumen HTML5
- `<html>` adalah elemen root (akar) dari semua konten HTML
- `<head>` menyimpan informasi meta, judul halaman, dan link CSS/JS
- `<body>` berisi semua konten yang akan dilihat pengunjung

## Tag dan Elemen

HTML menggunakan tag seperti `<h1>`, `<p>`, `<a>`, dan lainnya. Tag biasanya punya **pasangan**:

```html
<p>Ini paragraf</p>
```

Tag pembuka `<p>` dan tag penutup `</p>`. Ada juga yang **tidak perlu tag penutup**, seperti:

```html
<br /> <!-- Baris baru -->
<img src="gambar.jpg" alt="Gambar keren" />
```

## Komentar dalam HTML

Komentar berguna untuk memberi catatan dalam kode:

```html
<!-- Ini adalah komentar -->
<p>Kode di atas tidak akan terlihat di halaman web</p>
```

## Latihan

2. Tulis struktur HTML dasar seperti contoh di atas
3. Tambahkan satu heading dan satu paragraf tentang diri kamu
4. Coba buka file-nya di browser
