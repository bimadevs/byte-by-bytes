# Pengantar HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dasar sebuah halaman web. Bisa dibilang, HTML adalah "kerangka bangunan" dari semua website yang kamu kunjungi setiap hari.

Kursus ini cocok banget buat kamu yang baru mulai ngoding dan ingin memahami cara kerja halaman web dari nol. Jangan khawatir, nggak perlu jago matematika atau bisa ngoding dulu â€” kita mulai dari awal, bareng-bareng.

## Apa yang akan Anda pelajari

Dalam kursus ini, Anda akan belajar:

- Apa itu HTML dan kenapa dia penting
- Struktur dasar dokumen HTML
- Elemen-elemen umum seperti heading, paragraf, gambar, tautan, dan list
- Cara membuat formulir sederhana untuk input dari pengguna
- Tips dan best practice menulis HTML yang rapi dan mudah dibaca

## Prasyarat

Tidak ada prasyarat khusus. Bahkan jika Anda belum pernah ngoding sebelumnya, Anda tetap bisa mengikuti kursus ini.

## Proyek Akhir

Di akhir kursus, Anda akan membuat halaman profil sederhana yang bisa jadi bagian pertama dari portofolio Anda. Proyek ini akan menunjukkan bahwa Anda sudah bisa menulis HTML dengan baik.
