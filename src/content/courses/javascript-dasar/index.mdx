# Pengantar JavaScript

JavaScript adalah bahasa pemrograman yang paling populer di dunia dan menjadi tulang punggung dari hampir semua website modern. Kursus ini dirancang untuk pemula yang ingin memulai perjalanan mereka dalam pengembangan web dengan JavaScript.

## Apa yang akan Anda pelajari

Dalam kursus ini, Anda akan belajar:

- Dasar-dasar JavaScript dan sintaksisnya
- Variabel, tipe data, dan struktur kontrol
- Fungsi dan objek dalam JavaScript
- Manipulasi DOM untuk membuat website interaktif
- Penanganan event dan interaksi pengguna
- Konsep asinkron dengan Promise dan async/await

## Prasyarat

Tidak ada prasyarat khusus untuk mengikuti kursus ini. Namun, pemahaman dasar tentang HTML dan CSS akan sangat membantu.

## Proyek Akhir

Di akhir kursus, Anda akan membangun aplikasi web sederhana yang menerapkan semua konsep yang telah dipelajari. Proyek ini akan menjadi portofolio pertama Anda dan menunjukkan kemampuan JavaScript yang telah Anda kuasai.

## Instruktur

Kursus ini dibawakan oleh instruktur berpengalaman yang telah mengajar JavaScript selama lebih dari 5 tahun dan bekerja sebagai developer web profesional.

Selamat belajar dan selamat memasuki dunia pengembangan web dengan JavaScript!
