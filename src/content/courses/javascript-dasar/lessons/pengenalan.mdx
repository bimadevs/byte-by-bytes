# Pengenalan JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi, dinamis, dan diinterpretasi yang merupakan salah satu teknologi inti dari World Wide Web, bersama dengan HTML dan CSS. JavaScript memungkinkan Anda membuat halaman web interaktif dan merupakan bagian penting dari aplikasi web.

## Sejarah Singkat

JavaScript diciptakan oleh Brendan Eich saat bekerja di Netscape pada tahun 1995. Awalnya dinamakan "Mocha," lalu "LiveScript," dan akhirnya "JavaScript." Meskipun namanya mirip, JavaScript tidak berhubungan dengan bahasa pemrograman Java.

## Mengapa Belajar JavaScript?

Berikut beberapa alasan mengapa JavaScript sangat penting untuk dipelajari:

1. **Universalitas**: JavaScript dapat dijalankan di hampir semua perangkat yang memiliki browser web.
2. **Keserbaguraan**: Dapat digunakan untuk pengembangan frontend, backend (Node.js), aplikasi mobile, dan bahkan game.
3. **Komunitas Besar**: Memiliki ekosistem dan komunitas pengembang yang sangat besar.
4. **Lowongan Kerja**: Keterampilan JavaScript sangat dicari di industri teknologi.

## Alat yang Diperlukan

Untuk mulai belajar JavaScript, Anda hanya memerlukan:

- Browser web modern (Chrome, Firefox, Edge, dst.)
- Editor teks (VS Code, Sublime Text, dll.)

## Hello World di JavaScript

Mari mulai dengan contoh sederhana. Berikut adalah kode JavaScript pertama Anda:

```javascript
console.log("Hello, World!");
```

Kode di atas akan menampilkan teks "Hello, World!" di konsol browser.

## Cara Menyisipkan JavaScript ke HTML

Ada tiga cara utama untuk menyisipkan JavaScript ke dalam halaman HTML:

### 1. Internal JavaScript

```html
<script>
  console.log("Hello from internal JavaScript!");
</script>
```

### 2. External JavaScript

```html
<script src="script.js"></script>
```

Dengan file `script.js` berisi:

```javascript
console.log("Hello from external JavaScript!");
```

### 3. Inline JavaScript

```html
<button onclick="alert('Hello!')">Klik Saya</button>
```

## Kesimpulan

JavaScript adalah bahasa pemrograman yang kuat dan fleksibel yang dapat membuat halaman web Anda menjadi hidup. Dalam pelajaran-pelajaran berikutnya, kita akan mempelajari lebih dalam tentang sintaks dan fitur JavaScript.
