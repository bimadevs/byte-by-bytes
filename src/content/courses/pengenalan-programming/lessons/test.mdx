Sebelum kita mulai nulis kode beneran, kita perlu kenalan dulu sama *senjata-senjata* penting buat ngoding. Tenang, nggak butuh laboratorium rahasia atau komputer NASA kok. 😄

## 1. Text Editor / Code Editor

Ini kayak buku tulis digital tempat kamu nulis kode. Tapi bukan sembarang catatan—editor ini biasanya punya fitur bantu kayak:

- Highlight warna buat kode
- Auto-suggest atau auto-complete
- Bisa langsung run / debug kode

**Rekomendasi:**
- [VS Code](https://code.visualstudio.com/) (gratis, ringan, banyak plugin)
- Sublime Text
- Atom (udah nggak di-maintain, tapi masih banyak yang pakai)

> VS Code itu kayak “Microwave segala bisa” buat programmer.

## 2. Browser

Kalau kamu belajar web development, browser jadi *teman akrab*. Kenapa? Karena semua yang kamu bikin akan tampil di situ.

**Rekomendasi:**
- Google Chrome (banyak fitur developer tools)
- Mozilla Firefox
- Edge (lumayan juga sekarang 😎)

Gunakan fitur **Inspect / DevTools** untuk:
- Lihat struktur HTML di halaman
- Cek dan edit CSS secara langsung
- Debug JavaScript

## 3. Terminal / Command Line

Terminal itu kayak “mode matrix”. Kamu bisa ngobrol sama komputer pakai teks langsung. Kelihatan serem, tapi powerful banget.

Kamu bisa:
- Navigasi folder
- Jalankan program
- Install tools dan package

Contoh perintah:
```bash
cd project-saya
code .
npm install
```

> Jangan takut terminal, lama-lama malah cinta 😍

## 4. Browser + Internet

Yes, belajar coding = bersahabat dengan Google. Bahkan programmer senior pun tiap hari Googling.

Website yang sering dipakai:
- [Stack Overflow](https://stackoverflow.com/)
- [MDN Web Docs](https://developer.mozilla.org/)
- [W3Schools](https://www.w3schools.com/)
- ChatGPT (iyain aja 😆)

## 5. Ekstensi Bantu (Opsional)

Di VS Code kamu bisa install ekstensi seperti:
- Prettier (biar kode kamu rapi otomatis)
- Live Server (buat lihat web kamu langsung di browser)
- GitLens (buat kerja tim)

## Kesimpulan

Untuk mulai belajar ngoding, kamu cuma butuh:
- Text editor (kayak VS Code)
- Browser (kayak Chrome)
- Sedikit keberanian buat buka terminal 😄

Kita siap tempur! Di materi selanjutnya, kita akan mulai dari HTML — pondasi utama sebuah web.
