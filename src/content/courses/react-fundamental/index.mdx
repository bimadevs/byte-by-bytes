# Pengantar React

React adalah pustaka JavaScript yang populer untuk membangun antarmuka pengguna (UI) yang interaktif dan reaktif. Dikembangkan oleh Facebook, React telah menjadi salah satu teknologi frontend paling populer di dunia.

## Apa yang akan Anda pelajari

Dalam kursus ini, Anda akan mempelajari:

- Konsep dasar React seperti komponen, props, dan state
- Mengelola state dengan useState, useEffect, dan hook lainnya
- Routing dengan React Router
- Mengelola formulir dan validasi di React
- Mengonsumsi API dan mengelola data
- Menerapkan styling di React dengan CSS-in-JS
- Praktik terbaik dan pola desain React
- Membangun aplikasi lengkap dengan React

## Prasyarat

Untuk mengikuti kursus ini, Anda harus memiliki pemahaman dasar tentang:
- HTML, CSS, dan JavaScript
- Konsep ES6 seperti arrow functions, destructuring, dan modules
- Git dan npm (dasar-dasarnya saja)

## Proyek Kursus

Sepanjang kursus, Anda akan membangun beberapa proyek mini dan satu proyek akhir berupa aplikasi daftar tugas (todo list) lengkap dengan fitur:

- Menambah, mengedit, dan menghapus tugas
- Menyaring dan mencari tugas
- Melacak progres dan statistik
- Menyimpan data secara persisten dengan API

## Instruktur

Kursus ini dibawakan oleh developer React berpengalaman yang telah bekerja dengan React sejak versi awal dan telah membangun berbagai aplikasi berskala besar menggunakan React.
